# UAS-Based Environmental Disturbance Assessment
### WireGrass Gravel Pit | Alabama

## 📌 Project Overview

This project evaluates environmental disturbance and terrain characteristics at the WireGrass Gravel Pit in Alabama using unmanned aerial system (UAS) imagery, LiDAR-derived elevation products, and ArcGIS Pro.

The objective was to identify disturbed areas, quantify vegetation loss, generate terrain products, and demonstrate how drone-based mapping can support environmental monitoring and land management.

---

## 🎯 Objectives

- Produce a high-resolution orthomosaic
- Generate a Digital Surface Model (DSM)
- Generate a Digital Terrain Model (DTM)
- Create hillshade and slope products
- Assess environmental disturbance
- Quantify vegetation coverage
- Demonstrate GIS workflows for UAS data

---

## 🛰️ Software & Tools

- ArcGIS Pro
- Pix4Dmapper
- DJI Matrice 300 RTK
- LiDAR
- Microsoft Excel

---

## 📊 Data

- UAS Imagery
- LiDAR Point Cloud
